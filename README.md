# Credit Card Fraud Detection

## Overview
This project uses Machine Learning to detect fraudulent credit card transactions.

## Models Used
- Logistic Regression
- Random Forest Classifier

## Dataset
- Total Transactions: 284,807
- Fraud Transactions: 492
- Normal Transactions: 284,315

## Results

| Metric | Logistic Regression | Random Forest |
|----------|----------|----------|
| Accuracy | 99.89% | 99.95% |
| Precision | 82.76% | 97.37% |
| Recall | 48.98% | 75.51% |
| F1 Score | 61.54% | 85.06% |

## Conclusion
Random Forest outperformed Logistic Regression and was selected as the final model.
